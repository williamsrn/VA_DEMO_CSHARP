To retrieve the repository to your local machine, click the 'Download ZIP' button on the lower left hand side of the webpage.  After the download completes, extract the file to your desktop or another location on your computer.  Open the extracted folder where you will then see the Visual Studio solution file VA Test1.sln.  You will need a minimum of Visual Studio 2010 to run the file.  If you have GitHub Desktop or GIT on your local machine, you can also clone the repository and then access VA Test1.sln.  For further instructions on Git/GitHub and cloning a repository, please refer to the associated documentation for the respective product.

 

With the project open in Visual Studio, you can easily build and run the solution locally on your computer by selecting the 'Debug' menu at the top of the screen, and then 'Start Without Debugging'.  Alternatively you can select ctrl+F5 as a shortcut to build and run the solution.  You should then see the project open in your web browser.

 

Once the project is running in your browser, you will be presented with a welcome page.  At the top of the page is a series of links to the various pages of the web site: Home, About, Contact, and Manage Demographics.

 

The Welcome Page

 

The Home link will return the user to the Welcome page from anywhere in the site.

 

The About link will direct the user to an About page.  This page provides details about the web site and some of the underlying technology in use.

 

The Contact link will present the user with easy access to my personal contact info.

 

Finally, the Manage Demographics link is where all the goodies are.  Clicking this link will present a page to manage demographic info for a series of individuals.  The user will be presented with a list of individuals that were previously entered.  If there are no previously stored records, a message will be displayed indicating there are no records to display.

Technical Documentation:

Purpose:

This web application provides user management and storage of personal demographic information for an unlimited number of individuals.  It was developed using C# ASP.NET. 

 

 

Data:

Individual records are stored in a Microsoft SQL Server database, and a web-based user interface is used for CRUD record management.  Users can enter new records, view details of individual records, edit record information, and delete records from the database.  The database consists of a single table named People.  Each record in the People table relates to a single persons demographic information.  The table consists of four columns, fName, lName, address, and email, which correspond to persons First Name, Last Name, Address, and Email, respectively.  CRUD functionality for managing the individual records is presented to the user via an easy-to-use table based format.

 

 

Maintenance:

This is a web-based application and was designed using the MVC architecture and the Entity Framework.   The backend utilizes a MS SQL Server database and the web-based frontend utilizes the Bootstrap framework for responsive web design, with customized javascript and css for an enhanced user experience (UX).
